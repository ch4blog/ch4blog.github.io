# ch4blog 前端風格說明

這個網站的視覺定位是「可愛系插畫風 + 馬卡龍流行感」，並加入 Neo‑brutalism 的厚邊框與硬陰影，營造藝術生的手帳與貼紙感。整體氣氛偏夢幻、療癒、活潑。

## 風格關鍵字
- Kawaii / 可愛系插畫風
- Pastel Pop / 馬卡龍色
- Sticker / 手帳貼紙感
- Neo‑brutalism / 厚邊框 + 硬陰影
- 童趣、溫柔、輕快

## 色彩系統（取自 styles.css）
- 背景：天空藍＋奶油米色的漸層
- 主色：#ff7aa2（berry）
- 輔色：#ffc6b8（peach）、#8bd7ff（sky）、#b7f5e0（mint）、#ffe57a（lemon）
- 描邊：#1b1b1b（ink）
- 文字：#1b1b1b（主文），#5f5a58（副文）

## 字體與排版
- 標題：Fredoka（圓潤、童趣、插畫感）
- 內文：Noto Sans TC（易讀、乾淨）
- 版面：大間距、圓角卡片、區塊清晰，視覺像畫本分頁

## 元件與視覺語彙
- 卡片：白底＋厚描邊＋硬陰影（像貼紙）
- 按鈕：圓角膠囊、立體陰影、滑過位移
- 標籤 / Badge：彩色小貼紙感，帶邊框
- 作品框：棋盤格底＋圓角＋虛線框，像插畫本的預留框
- 裝飾：星星、愛心、泡泡等 doodle，強化可愛氛圍

## 版面結構
- Hero：大標題 + 貼紙感 badge + 行動按鈕
- 作品牆：多欄卡片展示（圖 + 標題 + 備註）
- 練習日誌：線條感紙張背景，像手帳頁
- 學習路徑：清單式卡片，偏「課表」感
- 關於 / 聯絡：簡潔模組化區塊

## 動效
- 進場：float-in（下方滑入）
- 裝飾：wiggle（緩慢擺動）
- 互動：hover 位移 + 陰影增強

## 圖片建議
- 風格：高飽和、乾淨邊緣、插畫或水彩風都適合
- 比例：4:3 或 16:10
- 留白：建議保留一點白邊，讓貼紙感更明顯

## 使用建議
- 保持色彩飽和但不刺眼，避免過多深色區塊
- 每區塊內容別太滿，保留呼吸感
- 若改成偏成熟風格，可減少 doodle、降低飽和度、改成柔霧陰影

## 專案結構（已拆頁）
- `index.html`：首頁（Hero + 精選作品 + 最新日誌）
- `works.html`：作品牆（完整作品列表）
- `journal.html`：練習日誌
- `learning.html`：學習計畫
- `about.html`：關於
- `contact.html`：聯絡
- `styles.css`：全站共用樣式
- `app.js`：共用導覽 active 樣式
